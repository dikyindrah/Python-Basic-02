Catatan:

mysql-connector

mysql-connector adalah library python dari MySQL 
yang menyediakan segala keperluan untuk berkomunikasi 
dengan database. 

mysql-connector adalah library pyhton yang 
paling umum untuk mengangani ini karena mudah 
untuk digunakan.

Untuk menginstal mysql-connector pada python kita 
bisa menggunakan manajemen paket yang ada pada 
pyhton, Kita hanya perlu memberikan perintah
'pip install mysql-connector-python' pada command
prompt atau terminal.

Setelah paket terinstal, selanjutnya kita bisa menguji
dengan mencoba mengakses database.

Contoh:

import mysql.connector

mydb = mysql.connector.connect(
    host = 'localhost',
    username = 'root',
    password = '',
    database = 'dbtest'
)

if mydb.is_connected():
    print('berhasil terhubung ke database')
