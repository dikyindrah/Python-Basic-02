Exception Handling

Adanya kesalahan (error) yang terjadi pada program saat 
dijalankan (runtime) dapat menyebabkan program itu berhenti
dan menimbulkan pengecualian. Pengecualian adalah objek
yang mewakili kesalahan (error).

Kesalahan tersebut bisa terjadi karena beberapa faktor
misalnya seperti kesalahan input atau jenis data yang 
di proses tidak sesuai. Untuk mengatasi kondisi seperti 
ini kita bisa menggunakan Exception Handling.

Exception Handling atau Penanganan Pengecualian adalah
cara yang digunakan untuk menangani kesalahan (error) 
pada program yang sedang dijalankan. Jadi dengan ini
kita bisa membuat program kita tetap berjalan meskipun
terjadi kesalahan (error).

Pada python exception handling bisa dilakukan dengan
menggunakan kata kunci (keyword) try, except, finally
else, dan rise. Dimana setiap kata kunci ini memiliki fungsi 
yang berbeda-beda.


try dan except

try digunakan untuk menguji kode program dan except
digunakan untuk menangani kesalahan (error) pada 
kode program yang diuji.

finally

finally akan menjalankan kode program dalam kondisi 
apapun baik saat terjadi kesalahan maupun saat tidak
terjadi kesalahan.

else

else digunakan untuk menambahkan pernyataan lain yang 
akan di eksekusi jika tidak ada kesalahan (error) yang 
terjadi.

rise

rise digunakan untuk menentukan jenis kesalahan apa yang 
akan dimunculkan, termasuk teks yang akan dicetak kepada 
pengguna.
