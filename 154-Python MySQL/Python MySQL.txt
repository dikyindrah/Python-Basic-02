Python MySQL

Kita bisa membuat aplikasi dengan bahasa pemrograman
python yang terhubung dengan database. Untuk membuat
ini kita perlu menginstal library yang menyediakan 
keperluan untuk berinteraksi dengan database.

Python menyediakan berbgai macam library untuk 
keperluan ini misalnya seperti mysql-connector, 
sqlalchemy, sqlite3, dan psycopg2.
