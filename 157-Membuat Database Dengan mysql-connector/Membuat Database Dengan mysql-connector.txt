Catatan:

Membuat Database Dengan mysql-connector

Untuk membuat database kita bisa menggunakan fungsi
mysql.connector.connect() untuk membuat koneksi 
ke database server dan cursor() untuk mengeksekusi
query database.

Contoh:

import mysql.connector
from mysql.connector import Error

try:
    with mysql.connector.connect(
        host = 'localhost',
        username = 'root',
        password = ''
    ) as koneksi:
        create_db_query = 'CREATE DATABASE db_profesi'
        show_db_query = 'SHOW DATABASES'
        with koneksi.cursor() as cursor:
            cursor.execute(create_db_query)
            print('Database berhasil dibuat\n')
            print('Daftar database:\n')
            cursor.execute(show_db_query)
            for db in cursor:
                print(db)
except Error as e:
    print(f"Database gagal dibuat, Error: {e}")


Output: 

Database berhasil dibuat

Daftar database:

('db_profesi',)
('dbbangundatar',)
('information_schema',)
('mysql',)
('performance_schema',)
('phpmyadmin',)
('test',)
