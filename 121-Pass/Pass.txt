Catatan:

Pass

Pass bisa kita gunakan untuk membuat pernyataan kosong. 
Misalnya saat kita membuat sebuah fungsi namun kita 
belum memikirkan apa yang harus dilakukan fungsi 
tersebut, kita tidak bisa mengosongkan fungsi itu
begitu saja karena interpreter python akan menganggap 
hal tersebut sebagai sebuah kesalahan. 

Oleh kerena itu kita bisa menggunakan pass untuk membuat
pernyataan kosong pada fungsi tersebut untuk memberi
tahu interpreter python bahwa kita memang sengaja 
mengosongkannya.

Contoh:

# Pernyataan kosong pada fungsi
def penjumlahan(x, y):
    pass

# Pernyataan kosong pada penanganan pengecualian
try:
    pass
except:
    pass
